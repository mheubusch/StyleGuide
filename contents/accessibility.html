Accessibility

Like localization, accessibility is an important aspect of Firefox copy creation. Similarly, writing well for the web – short sentences, clear hierarchy, non-ambiguous language – supports accessibility efforts. Additional considerations include the use of descriptive and meaningful alt text. You can learn more from the World Wide Web Consortium about design standards for users who rely on assistive technologies.

It is also important to be fair and respectful when we talk about all Firefox users, regardless of how they use our products to access the Web. Read advice from the ADA Nation Network on removing bias in accessibility language.
